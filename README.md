# 📐 UML & BPMN Examples: Диаграммы для проектирования

Этот репозиторий содержит коллекцию **UML-диаграмм** и **BPMN-моделей**, созданных в рамках учебных и практических проектов. Здесь собраны примеры моделирования бизнес-процессов, проектирования баз данных и документирования пользовательских сценариев.

## 📁 Содержание

### 🔄 BPMN-диаграммы (Business Process Model and Notation)

Файл `BPMN_diagrams.pdf` содержит три диаграммы, описывающие ключевые процессы барбершопа:

| Процесс | Владелец | Краткое описание |
|---------|----------|------------------|
| **BPHair4. Обратная связь** | Руководитель | Написание отзыва о проделанной работе мастера и анализ этой обратной связи руководителем |
| **BPHair3. Оказание услуги** | Мастер | Оказание услуги с момента напоминания мастеру о приближающейся записи и до момента ее полного оказания клиенту |
| **BPHair6. Финансы и учёт** | Бухгалтер | Получение данных об оплате, расчет ЗП, подготовка и отправка отчетов регуляторам |

### 🧱 Моделирование базы данных

Диаграммы, описывающие структуру и связи сущностей информационной системы.

| Файл | Назначение |
|------|-------------|
| `UML BSA09-04_2026-04-05T07_20_12.958Z_2.dbml` | Модель данных в формате DBML |
| `UML-диаграмма.png` | Графическое представление ER-диаграммы / диаграммы классов БД |

### 🔄 Диаграммы состояний (State Machine)

Визуализация жизненных циклов ключевых объектов системы.

| Файл | Описание процесса |
|------|-------------------|
| `Диаграмма состояний - оплата.png` | Жизненный цикл платежа: инициализация, подтверждение, отклонение, возврат |
| `Оплата.drawio` | Исходник диаграммы оплаты в формате Draw.io |
| `Диаграмма состояний - доставка.png` | Жизненный цикл заказа: сборка, транспортировка, вручение, отмена |
| `Доставка.drawio` | Исходник диаграммы доставки в формате Draw.io |

## 🎯 Назначение

Диаграммы могут быть полезны для:
- Понимания структуры и связей сущностей при проектировании баз данных.
- Анализа переходов и состояний объектов сложных систем.
- Документирования архитектуры и пользовательских сценариев.
- Визуализации и оптимизации бизнес-процессов (BPMN).
- Подготовки к собеседованиям и техническим заданиям.

## 🛠 Инструменты

Все исходные файлы созданы с помощью:
- [Draw.io](https://app.diagram.net/) — для BPMN и State Machine диаграмм.
- [DrawDB](https://www.drawdb.app/) — для визуализации схемы БД.
